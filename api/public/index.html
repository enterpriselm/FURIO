How Does This Tool Work?

When you upload an image and set thresholds, our tool uses a mathematical method called the Fast Fourier Transform (FFT).

FFT is a powerful technique in signal processing that breaks down an image into its frequency components, separating high-frequency elements from low-frequency ones.

This lets us create:

  • High-frequency masks: Highlight fine details, textures, and edges in the image.
  • Low-frequency masks: Reveal broader patterns or smoother regions of the image.

Understanding the Parameters

1. Image Upload: Upload any image, preferably in color (e.g., .jpg, .png). This is the image that will be processed to create the frequency masks.

2. Low Threshold: Controls the lower limit for frequency filtering. Adjusting this threshold allows you to control how much low-frequency content is retained in the low-frequency mask. Threshold Range [0,1].

3. Upper Threshold: Controls the upper limit for frequency filtering. Adjusting this threshold lets you control how much high-frequency content is retained in the high-frequency mask. Threshold Range [0, 1].
